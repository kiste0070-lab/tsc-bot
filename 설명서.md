# TSC 트레이닝 센터 봇 설명서 (TTS 통합형)

듣기 기능이 추가된 완벽한 **TSC 트레이닝 센터**입니다. TSC는 말하기 시험이지만, 상대방의 질문을 정확히 알아듣는 '청취력'이 전제되어야 하므로 TTS(Text-to-Speech) 기능을 통해 학습 효과를 극대화했습니다.

---

### 1. 주요 기능
*   **월간 문제 계획 시스템**: `Monthly_Plan/YYYY_MM.md` 파일에 매월 1일부터 말일까지 Part 2~6 문제를 날짜별로 미리 생성합니다. 월간 계획은 문제가 절대 겹치지 않도록 설계되었으며, 기존 문제와 중복 검사를 통해 재생성합니다.
*   **일일 문제 출제**: GitHub Actions 실행 시 오늘 날짜에 해당하는 문제를 `Monthly_Plan`에서 읽어 텔레그램으로 발송합니다.
*   **지능형 예약 실행**: 지정된 시간까지 대기했다가 자동으로 수업을 시작합니다.
*   **리소스 최적화**: 하루 한 번 수업이 끝나면 프로세스가 자동으로 종료되어 백그라운드 리소스를 점유하지 않습니다.
*   **음성 자동 발송**: 봇이 중국어로 질문을 던질 때, 해당 텍스트를 음성으로 변환하여 텔레그램 음성 메시지(.voice)로 즉시 전송합니다.
*   **성조 학습 최적화**: 원어민에 가까운 구글 TTS 엔진(`zh-CN`)을 사용하여 정확한 성조와 발음을 들으며 연습할 수 있습니다.
*   **Gemini 기반 피드백**: 사용자의 답변을 분석하여 상세한 피드백(교정 및 병음)을 제공합니다. 미답변 부분에 대해서는 HSK 1~4급 단어 중심의 예시 답변을 제공합니다.
*   **오답 관리**: `wrong_notes/YYYYMM_wrong_notes.md` 형식으로 월별 오답노트를 자동 기록합니다. 날짜별로 나의 답변과 AI 첨삭 내용이 쌓이며, GitHub Actions 실행 시 repo에 자동으로 커밋됩니다.

### 2. 설치 및환경 설정

#### 필수 요구 사항
*   **Python 3.10 이상** 권장
*   필수 라이브러리 설치:
    ```bash
    pip install gTTS python-telegram-bot google-genai python-dotenv
    ```

#### 환경 변수 설정 (`.env`)
프로젝트 폴더 내에 `.env` 파일을 생성하고 아래 내용을 입력하세요.
```env
TELEGRAM_BOT_TOKEN=여러분의_텔레그램_봇_토큰
GEMINI_API_KEY=여러분의_제미나이_API_키
CHAT_ID=여러분의_텔레그램_ID
```

#### GitHub Actions Secrets 설정
GitHub repo → **Settings → Secrets and variables → Actions** 에서 아래 3개를 등록하세요.
| Secret 이름 | 설명 |
|---|---|
| `TELEGRAM_BOT_TOKEN` | 텔레그램 봇 토큰 |
| `GEMINI_API_KEY` | Gemini API 키 |
| `CHAT_ID` | 텔레그램 채팅 ID |

> ⚠️ **권한 설정 필수**: `Settings → Actions → General → Workflow permissions`에서 **Read and write permissions**를 선택해야 오답노트 자동 커밋이 가능합니다.

### 3. 사용 방법 및 문제 해결

#### 실행 방법
1.  **봇 실행**: 코드가 있는 폴더에서 `python main.py`를 실행합니다.
2.  **GitHub Actions**: 매일 한국 시간 오전 11:59에 자동 실행됩니다. 수동 실행은 GitHub → Actions 탭 → `Daily TSC Lesson` → **Run workflow** 버튼을 클릭하세요.

#### 월간 문제 계획 구조
```
Monthly_Plan/
  ├── 2026_03.md   ← 2026년 3월 문제 계획
  ├── 2026_04.md   ← 2026년 4월 문제 계획
  └── ...
```
파일 내부는 날짜별(`### YYYY-MM-DD`)로 구분되며, 각 날짜마다 Part 2~6 문제가 1개씩 포함되어 있습니다. 월간 계획은 이미 존재하면 생성을 건너뜁니다.

#### 오답노트 구조
```
wrong_notes/
  ├── 202603_wrong_notes.md   ← 2026년 3월 오답
  ├── 202604_wrong_notes.md   ← 2026년 4월 오답
  └── ...
```
파일 내부는 날짜별(`## YYYY-MM-DD`)로 구분되며, 나의 답변과 AI 첨삭이 함께 기록됩니다.

#### 수업 진행 방식
1. 수업이 시작되면 Part 2~6 문제가 한 번에 주어집니다.
2. 본인이 답변하고 싶은 부분만 선택하여 답변할 수 있습니다. (예: `3부분 : (내 답변)`)
3. 답변한 부분에 대해서는 AI 첨삭을 제공합니다.
4. 답변하지 않은 부분에 대해서는 HSK 1~4급 단어 중심의 2문장 예시 답변을 제공합니다.
5. 모든 답변 후 첨삭과 예시 답변이 제공되면 수업이 종료됩니다.
6. **"수업종료"**라고 입력하면 모든 부분에 대한 예시 답변을 제공하고 수업이 종료됩니다.

#### 문제 해결 (Troubleshooting)
*   **RuntimeError (Event Loop)**: 만약 "This event loop is already running" 에러가 발생한다면, `main.py`의 `main()` 함수가 동기(sync) 방식으로 작성되었는지 확인하세요. 최신 코드에서는 이 문제가 해결되어 있습니다.
*   **gTTS 에러**: 인터넷 연결 상태를 확인하세요. gTTS는 구글 서버에 접속하여 음성을 생성합니다.
*   **문제를 찾을 수 없음**: `Monthly_Plan` 폴더에 해당 월의 계획 파일이 없거나 손상되었을 수 있습니다. 파일을 삭제하면 다음 실행 시 자동으로 재생성됩니다.

### 🎧 학습 팁
1.  **실전 모의고사 방식 (Part 2~6 연속 훈련)**: 수업이 시작되면 2부분부터 6부분까지 총 5개의 문제가 한 번에 주어집니다. 실전 시험을 치르듯이 문제를 쭉 확인하세요.
2.  **선택적 답변하기**: 본인이 답변하고 싶은 특정 파트만 골라서 아래 형식으로 답변을 보낼 수 있습니다. (예: `3부분 : (내 답변)  / 5부분 : (내 답변)`)
3.  **맞춤형 피드백**: AI 봇은 사용자가 답변을 적어 보낸 부분에 대해서만 꼼꼼하게 한국어 첨삭/교정 및 오답노트를 작성해줍니다. 답변하지 않은 부분에는 예시 답변을 제공합니다.
4.  **눈보다 귀를 먼저**: 텍스트 메시지를 보기 전에 음성 메시지를 먼저 재생해서 들어보세요. TSC 시험장의 실제 환경과 비슷하게 훈련할 수 있습니다.
5.  **쉐도잉 연습**: 봇이 보낸 음성 메시지를 듣고 그대로 따라 말해본 뒤, 자신의 답변을 입력하면 말하기 실력 향상에 큰 도움이 됩니다.
6.  **오답노트 활용**: 정기적으로 오답 노트를 확인하고 봇에게 해당 문장들을 다시 읽어달라고 요청해 보세요.
7.  **월간 계획 활용**: 매월 새로운 문제로 학습할 수 있으며, 문제가 겹치지 않도록 설계되어 반복 학습에 최적화되어 있습니다.
